MySQL es una herramienta de administración de bases de datos de código abierto que emplea lenguaje de consulta estructurado (SQL) para acceder, añadir y gestionar el contenido de una base de datos. Al ser de código abierto, es una herramienta muy utilizada por desarrolladores de software, formando así la estructura principal de la mayoría de los sistemas operativos más conocidos.

La mayoría de sistemas de gestión de contenido modernos (como WordPress) lo usan para almacenar datos. De esta forma, cuando se hace una modificación en el CMS, ésta se registra en la base de datos MySQL, y luego cada vez que un visitante llega al sitio, se le muestra el contenido de esa base de datos.

De seguro el término "ancho de banda" te suena familiar, gracias a tu proveedor de Internet. Podemos definir este término como la cantidad de datos que pasan por una conexión a Internet.

Cuando un usuario quiere visitar tu página web, le envía una solicitud al servidor. A su vez el servidor responde y le muestra la información que hay en la página al usuario. Mientras más personas entren a tu sitio más ancho de banda consumirás, hasta llegar a un punto en el que estarás a plena capacidad.

Cuando muchos usuarios intentan ingresar al mismo tiempo a tu sitio web, la tubería se llena y el sitio puede cargar lento o incluso puede dejar de funcionar. Esto es lo que se encuentra determinado por el ancho de banda del alojamiento web.

Podría decirse que tu página web es una colección de archivos y base de datos, y esta información se le muestra a los visitantes en los diferentes navegadores web. Hostinger ofrece espacio para que tanto la página web como las bases de datos sean almacenadas en tu host.

Todo sitio web consta de un dominio y un hosting web. El dominio es como tu dirección y el hosting web como el edificio físico.

Cuando estás navegando por internet y escribes https://www.google.com.ar, por ejemplo, le estás diciendo a tu navegador web que el dominio es hostinger.com.ar y este buscará cuál es su hosting web para cargar el contenido del mismo.
